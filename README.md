# Prode App 🏆

Un ecosistema avanzado de predicciones deportivas (Prode) diseñado para una experiencia competitiva y social de alto impacto.

## 🚀 Arquitectura Técnica

- **Frontend**: React 19 + TypeScript + Vite.
- **Estilo**: Tailwind CSS 4 para una UI moderna y performante.
- **Backend & Auth**: Supabase (PostgreSQL, Auth, Storage).
- **Animaciones**: Framer Motion.
- **PWA**: Soporte nativo para instalación en dispositivos móviles.

## 🛠️ Configuración del Proyecto

### Requisitos Previos

- Node.js (v18+)
- NPM o PNPM

### Instalación

```bash
npm install
```

### Entorno de Desarrollo

Crea un archivo `.env` en la raíz con las siguientes variables:

```env
VITE_SUPABASE_URL=tu_url_de_supabase
VITE_SUPABASE_ANON_KEY=tu_anon_key
VITE_API_FOOTBALL_KEY=tu_api_key_de_football
```

Para iniciar el servidor de desarrollo:

```bash
npm run dev
```
